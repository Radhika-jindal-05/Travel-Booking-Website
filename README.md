# 🌍 Travel Booking Website

A responsive and visually appealing **Travel Booking Website** built using **HTML, CSS, and JavaScript**.  
It allows users to explore destinations, view travel videos, and make online bookings easily.

---

## 🚀 Features

- 🏖️ Beautiful landing page with travel video background  
- 📅 Interactive booking form for date and destination selection  
- 📸 Image and video gallery showcasing top destinations  
- 📱 Fully responsive design for all screen sizes  
- 🌐 Simple and clean navigation bar  
- 💬 Contact form for inquiries  

---

## 🛠️ Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and layout (including Flexbox/Grid)  
- **JavaScript (Vanilla JS)** – Interactivity  
- **Font Awesome** – Icons  
- **Google Fonts** – Typography
